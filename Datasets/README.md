# Coding Samurai – Data Science Internship Projects

This repository contains all tasks and projects completed as part of the Coding Samurai Data Science Internship.  
The goal of this internship was to gain practical experience in Data Analysis, Machine Learning, Time Series Forecasting, and Natural Language Processing.

---

## Projects Included

### Project 1 – Sales Data Analysis
Performed exploratory data analysis on retail sales data using visualizations and descriptive statistics.

### Project 2 – Linear Regression
Built a predictive linear regression model to analyze income trends using numerical datasets.

### Project 3 – Titanic Exploratory Data Analysis
Analyzed the Titanic dataset to identify survival trends, feature relationships, and missing values.

### Project 4 – Titanic Logistic Regression
Developed a classification model to predict passenger survival using logistic regression techniques.

### Project 5 – Time Series Forecasting
Implemented ARIMA time series forecasting on Apple stock price data to predict future trends.

### Project 6 – Sentiment Analysis (NLP)
Applied Natural Language Processing techniques to classify text sentiment using polarity scoring.
