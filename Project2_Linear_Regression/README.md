# Linear Regression – Project 2

## Objective
Build a regression model to predict income trends based on numerical features.

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
Government household income dataset (sample used).

## Tasks Performed
- Data cleaning and numeric conversion
- Train-test split
- Linear Regression model building
- Model evaluation using R² and MSE
- Regression visualization

## Outcome
Successfully built a predictive regression model and evaluated its performance using statistical metrics.
